選擇排序法的原理

選擇排序是一種簡單的排序法。每次從未排序的元素中找到最小值，放在已排序部分的末尾。通過這樣的選擇過程，整個陣列逐步建立了有序序列。

基本步驟：

  1. 從未排序的元素中找到最小值
  2. 將最小值與已排序部分的第一個元素交換
  3. 移動已排序部分的邊界，向右移動一位
  4. 重複此過程，直到所有陣列都排序完成

JavaScript 實現：

function selectionSort(arr) { const n = arr.length; let comparisons = 0; let swaps = 0; for (let i = 0; i < n - 1; i++) { let minIndex = i; for (let j = i + 1; j < n; j++) { comparisons++; if (arr[j] < arr[minIndex]) { minIndex = j; } } if (minIndex !== i) { [arr[i], arr[minIndex]] = [arr[minIndex], arr[i]]; swaps++; } } return { comparisons, swaps }; }
⏱️ 時間與空間複雜度

最壞情況

O(n²)

陣列完全反序排列時需要最多的比較

平均情況

O(n²)

隨機排列的陣列平均需要約 n²/2 次比較

最好情況

O(n²)

即使陣列已排序，仍需進行完整的 n²/2 次比較

空間複雜度

O(1)

只需常數空間，In-place 排序

⚖️ 優點與缺點

✓ 優點

  • 實現簡單易懂
  • 空間複雜度低 (O(1))
  • 交換次數最少
  • 對小規模陣列有效

✗ 缺點

  • 時間複雜度高 (O(n²))
  • 對大規模陣列不適用
  • 比較次數無法減少
  • 實務中很少使用
