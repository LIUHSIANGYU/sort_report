🔄 氣泡排序法 (Bubble Sort)

視覺化演示與複雜度分析

📖 操作原理

核心概念

氣泡排序法是最直觀的比較排序演算法。每次掃描陣列，逐一比較相鄰的兩個元素，若左邊的元素大於右邊的元素，就將它們互換。每完成一輪掃描，最大的元素就會「浮」至陣列末尾，就像泡泡一樣浮起。

掃描過程

透過重複掃描，每次比較相鄰元素，若發現順序錯誤就互換。經過多輪掃描後，最大的元素逐步移到末尾，整個陣列最終達到有序狀態。

📖 演算法說明

氣泡排序法的原理

氣泡排序是最簡單的排序演算法。它通過不斷比較相鄰的元素，將較大的元素交換到右邊，使得大的數值像氣泡一樣逐步「浮起」到陣列的右端。

演算法偽代碼：

BUBBLE-SORT(A)
n = A.length
for i = 1 to n-1
for j = 1 to n-i
if A[j] > A[j+1]
swap A[j] with A[j+1]
⏱️ 時間與空間複雜度

最壞情況

O(n²)

陣列完全反序排列時需要最多的比較和交換

平均情況

O(n²)

隨機排列的陣列需要約 n²/4 次比較

最好情況

O(n)

陣列已排序時，只需 n-1 次比較

空間複雜度

O(1)

只需常數空間，In-place 排序

⚖️ 優點與缺點

✓ 優點

  • 實現簡單易懂
  • 空間複雜度低 (O(1))
  • 穩定排序
  • 對小規模陣列有效

✗ 缺點

  • 時間複雜度高 (O(n²))
  • 對大規模陣列不適用
  • 效率遠低於其他演算法
  • 實務中很少使用
